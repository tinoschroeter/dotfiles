## vim
* gg=G           # formating code
* :set nu!       # hide line Number
* :50            # go to line Number 50
* /search        # search type n for next
* :%s/this/that/ # replace thist with that 
* :66,70s/^/#    # comment line 66 - 70
* :66,70s/^#/    # and reverse
